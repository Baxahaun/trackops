El contexto lo es todo

La IA se encarga del código.
TrackOps del proyecto.

Cursor, Copilot o Windsurf generan código a velocidad de vértigo. Pero cuando el proyecto crece, la IA pierde el hilo: olvida lo ya hecho, ignora las prioridades y empieza a contradecirse. El contexto se corrompe.

TrackOps mantiene una fuente de verdad determinista en archivos Markdown sincronizados. Cada trackops sync traduce el estado real del proyecto en instrucciones que cualquier agente puede consumir directamente. No dejes que tu IA adivine lo que tiene que hacer.

Contexto Determinista para tu IA

Tu agente sabe exactamente qué hacer. Siempre.

Cada vez que ejecutas trackops sync, el motor traduce el estado del proyecto en tres archivos que Claude, Cursor o Copilot leen como contexto de sistema:

  • task_plan.md El mapa de trabajo actual. La IA ve qué está pendiente y qué debe hacer.
  • progress.md El historial de lo ejecutado. Evita que la IA reescriba cosas terminadas.
  • findings.md Hallazgos, decisiones técnicas previas y bloqueos resueltos.
Ejemplo .cursorrules / CLAUDE.md 
## Control Operativo

Antes de iniciar cualquier tarea:
1. Lee `task_plan.md` para entender el estado.
2. Lee `progress.md` para el contexto.
3. Lee `findings.md` para conocer decisiones.
4. Usa `trackops status` para confirmar.

Al completar trabajo:
1. Ejecuta `trackops task complete `.
2. Ejecuta `trackops sync` para actualizar.

Metodología Opcional · 5 Fases

Metodología O.P.E.R.A.

OPERA es el protocolo de ejecución que convierte cualquier idea en un proyecto de IA predecible. Cinco fases con Definition of Done verificable: desde la visión hasta el despliegue continuo.

npx trackops opera install
O

Orquestar

Visión, datos y reglas de negocio.

Entregable: genesis.md
P

Probar

Conectividad, validación y viabilidad técnica.

Entregable: Test scripts
E

Estructurar

Construcción core en 3 capas e integración de SOPs.

Entregable: SOPs + Integración
R

Refinar

Auditoría de calidad y validación de outputs de la IA.

Entregable: Outputs validados
A

Automatizar

Despliegue continuo, smoke tests y triggers.

Entregable: Triggers + Deploy

Ecosistema de Skills

Las Skills son módulos de instrucciones especializadas que dotan a tu agente de protocolos específicos: commits semánticos, changelogs automáticos, inicialización de proyectos. Un comando. Instaladas. Activas.
